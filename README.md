## 这是什么

一款受 QQ 农场启发、但内核完全不同的网页游戏：**每种作物对应一个数学级数，每次收割拿到级数的下一项，部分和实时逼近极限**。

你会亲眼看着几何草莓的收成爬向 20、调和玉米永远在涨却涨得越来越慢、π葡萄正负横跳着逼近 10π —— 在种菜偷菜的过程中，不知不觉理解了**收敛、发散、极限**。

## 核心玩法：级数收割

种一次，**永不清空**，每次收割拿第 n 项，地块自动重新生长。

| 作物 | 级数 | 极限 | 体验 |
|---|---|---|---|
| 🍓 几何草莓 | `10·(½)ⁿ⁻¹` | 收敛 → 20 | 越收越少，摸不到顶 |
| 🌽 调和玉米 | `10/n` | 发散 → ∞ | 永远有得收，但越来越慢 |
| 🍉 交错西瓜 | `(-1)ⁿ⁺¹·10/n` | 收敛 → 10·ln2 | 正负横跳逼近 |
| 🎃 阶乘南瓜 | `100/n!` | 收敛 → 100(e-1) | 几次就撞顶 |
| 🍇 π葡萄 | `(-1)ⁿ⁺¹·40/(2n-1)` | 收敛 → 10π | 莱布尼茨逼近 π |
| 🌻 欧拉向日葵 | `10/n²` | 收敛 → 10·π²/6 | 巴塞尔问题 |

每块地都带一条**实时收敛曲线**，点开有大图，看部分和 Sₙ 一步步爬向虚线极限。

## 功能特性

- 用户系统（注册 / 登录，JWT）
- 级数作物种植 / 无穷收割 / 金币 / 等级 / 经验
- 收敛曲线可视化（卡片缩略图 + 详情大图 + 公式/极限展示）
- 好友系统：加好友、逛好友农场、偷菜、被偷记录
- 坐标纸主题界面

## 技术栈

- **前端**：Vue 3 + Vite（本仓库）
- **后端**：Node.js + Express（独立服务）
- **数据库**：MySQL
- **部署**：阿里云轻量服务器 + PM2

数学核心在 [`src/series.js`](src/series.js)，前后端共用同一套级数公式。

## 本地运行（前端）

```bash
npm install
npm run dev
```

然后访问 `http://localhost:5173`。前端会请求已部署的后端 API。
