📐 数理农场 · MathFarm

设计文档 v0.1 · 2026-06-15 · 一款把高数、金融、无穷级数种进土里的模拟经营游戏

「不是把数学贴在农场上，而是让数学成为农场的土壤本身。」

一、世界观 & 核心理念

玩家是一位「数学农夫」，在一片神奇土地上耕种。这里的作物不产生固定收益——每一次收割，都是一次数学运算的具现化。土地本身由不同的「数学性质」构成，季节影响公式参数，好友互动变成数学协作。

核心体验目标：玩家在不知不觉中理解了极限、收敛、复利、导数，然后某一天突然说「原来高中数学讲的就是这个意思！」

二、作物系统 — 无穷级数田

每种作物对应一个数学级数。每次收割拿到「第 n 项」，所有收割累计形成「部分和 Sₙ」，右侧实时显示逼近曲线。

作物级数公式极限体验感
🍓 几何草莓aₙ = 10·(½)ⁿ收敛 → 20头几次爆收，越收越少，体会「等比级数收敛」
🌽 调和玉米aₙ = 10/n发散 → ∞收益一直有但慢到让人怀疑，反直觉的发散
🍉 交错西瓜aₙ = (-1)ⁿ⁺¹·10/n收敛 → 10·ln2收成正负横跳，曲线左右震荡逼近目标
🎃 阶乘南瓜aₙ = 100/n!收敛 → 100(e-1)极快收敛，几次就撞顶，感受阶乘的恐怖
🍇 π 葡萄aₙ = 4·(-1)ⁿ/(2n-1)收敛 → π·金币莱布尼茨公式逼近π，收够了解锁「π 勋章」
🌻 欧拉向日葵aₙ = 1/n²收敛 → π²/6巴塞尔问题，收完显示「这就是欧拉的答案」
UI 核心：每块地旁边有一条实时折线图，横轴=收割次数，纵轴=部分和 Sₙ，虚线=理论极限。

三、土地系统 — 微积分地形

土地类型

📈 导数沃土
生长速度 = 当前作物产量的导数。产量越高长得越快，正反馈土地。
∫ 积分湿地
收割奖励 = 生长曲线下方面积（定积分）。种得越久爆发越强。
📉 极限荒地
初始贫瘠，每轮以 ε→0 速度改良，ε 挑战：误差内完成任务解锁升级。
🌀 泰勒温室
产量用泰勒展开近似，玩家升级「展开阶数」提升收益精度。

天气系统（微分方程驱动）

☀️ 晴天
标准生长 dy/dt = ky（指数增长）
🌧️ 雨天
连续降雨引入阻尼 dy/dt = ky - cy²（逻辑斯谛增长）
❄️ 雪天
生长暂停，「冰晶」道具解「冻结方程」解锁新作物
🌀 混沌风暴
洛伦兹吸引子随机事件，蝴蝶效应彩蛋

四、金融系统 — 数学经济层

🏦 复利银行

存入金币按 A = P·eʳᵗ 连续复利增长，利率 r 随市场情绪波动（布朗运动模拟）。

📊 期货市场

种下时锁定未来收购价，实际市价与合约价差决定盈亏，体验对冲与套保。

📉 波动率农场

价格服从 dS = μS·dt + σS·dW（几何布朗运动），波动越大产量越高但风险越大。

💹 组合种植

同时种多种作物，收益 = 加权平均，玩家调权重体验马科维茨投资组合理论。

🎰 衍生品（高级）

• 看涨期权：花小钱买未来固定价格收割的权利，Black-Scholes 公式定价显示在界面

• 套利机会：好友农场与自己作物价差 > 运输成本时，系统提示「套利窗口开启 3 分钟」

五、社交系统 — 数学协作

🤝 级数接力

你贡献奇数项，好友贡献偶数项，合并成合作级数，共享收敛奖励，合作比独种快 2 倍。

😈 偷菜升级版

偷走「第 n 项」，同时给对方留下「导数凭证」——对方能看到被偷次数与速率曲线。

🏆 排行榜

• 部分和榜：谁的 Sₙ 最接近理论极限

• 收敛速度榜：误差下降最快

• 财富榜：复利银行总资产

• 混沌榜：风暴天气存活最久

📬 数学挑战信

向好友发「用 10 次收割让 |Sₙ - L| < 0.1」，完成获双倍奖励。
